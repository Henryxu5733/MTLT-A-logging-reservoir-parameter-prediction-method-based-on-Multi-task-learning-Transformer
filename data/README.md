## Example Data Description

The file `example.csv` is a synthetic dataset created solely for demonstrating the data format and workflow.

### Features
- AC: Acoustic log
- CAL: Caliper log
- GR: Gamma ray log
- RT: True resistivity
- RXO: Flushed zone resistivity

### Targets
- PERM: Permeability
- POR: Porosity
- SW: Water saturation

⚠️ Note: This dataset does NOT contain any real industrial data.
